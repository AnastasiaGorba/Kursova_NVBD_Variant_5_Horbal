Курсова робота з дисципліни "Надвеликі бази даних"
Тема: Розробка сховища даних для предметної області "Виконання проектів" Варіант: 5 Виконала: Студентка групи ПМ-43, Горбаль Анастасія

Опис предметної області
Система призначена для аналізу ефективності виконання проектів, фінансових показників та завантаженості персоналу.

Обсяг даних: > 500 000 записів.
Період даних: 5 років.
Технології
DBMS: Microsoft SQL Server 2019
ETL: SQL Server Integration Services (SSIS)
OLAP: SQL Server Analysis Services (SSAS Multidimensional)
Reporting: SQL Server Reporting Services (SSRS)
Структура репозиторію
/Database - SQL-скрипти створення БД, таблиць та генерації даних.
/SSIS_Projects - ETL пакети для завантаження даних у Data Warehouse.
/SSAS_Projects - Багатовимірний OLAP-куб.
/SSRS_Projects - Аналітичні звіти (.rdl файли).
/Documentation - Звіт (PDF).
Реалізовані звіти (SSRS)
Список проектів по категоріях (Табличний звіт з фільтрацією).
Зарплатна відомість (Матричний звіт).
Аналіз завантаженості (Діаграми: стовпчикова, лінійна, кругова).
Дашборд керівника (KPI, індикатори, спарклайни).
Деталізація проектів (Drill-down звіт).

Як запустити
Розгорнути БД за скриптами з папки /Database.
Відкрити solution у Visual Studio.
Змінити Connection Strings на свій сервер.
Розгорнути (Deploy) куб SSAS.
Запустити звіти SSRS.
